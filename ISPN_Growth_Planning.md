1.3 ROI Calculator — Project-Specific Rules in CLAUDE.md

Architecture: React+TS (Vite) frontend + Python/FastAPI backend. No database. No auth.
ISPN Constants: $21.00 blended rate, 1.35x multiplier, 60% utilization, 35% turnover, 6-wk training ramp, 1.5x after-hours, 15% management overhead, 8% QA overhead, Genesys licensing for tech cost.
Conventions: TS strict mode. Decimal for money in Python, USD with commas. Pydantic models for all API contracts. Recharts for charts. SkillResult typed response envelope (per fastapi-patterns.md).
Testing: pytest backend, vitest frontend. Both must pass before marking done.

1.4 GCS Engine — Project-Specific Rules in CLAUDE.md

Architecture: Python CLI pipeline (pandas, numpy, openpyxl). No web server, no database.
Weights: volume_growth 25%, sla_degradation 20%, service_concentration 15%, bead_exposure 15%, utilization_headroom 10%, repeat_contacts 10%, contract_proximity 0%, seasonal_volatility 5%.
Scoring: Each signal 0-3. Composite weighted sum → 0-100. Tiers: green (>70), amber (40-70), red (<40).
Connector pattern: get_data() -> pd.DataFrame. Stubs swap to live via config.py import.
Testing: pytest. End-to-end: python -m src.main.
